# Document Management System - Using MEAN STACK

This is a web application project, where we can upload, search, and manage documents with version control.

## Tech Stack Used:
- **Frontend:** Angular 
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **File Handling:** Multer

## The features of this project are as follows:
- **User Authentication:** User authentications is provided using JWT.
- **Document Management:** Upload ,search, and update documents.
- **Responsive UI:** Built with Angular and Bootstrap

## Installation and setup

1. git clone https://github.com/Charan-tej-03/DocumentManagementSystem.git
2. cd DocumentManagementSystem
3. cd backend
4. npm install
5. npm start

In a new terminal:

6. cd frontend
7. npm install
8. ng serve
9. We can access the web-application through http://localhost:4200/
